# Rock vs Mine Prediction

## Overview
Machine Learning project that predicts whether a SONAR signal belongs to a Rock or a Mine.

## Dataset
- 208 samples
- 60 numerical features
- Binary classification

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Google Colab

## Workflow
1. Data Loading
2. Exploratory Data Analysis
3. Data Preprocessing
4. Train-Test Split
5. Feature Scaling
6. Logistic Regression
7. Model Evaluation
8. Prediction System

## Results
- Training Accuracy: 91.39%
- Testing Accuracy: 61.90%
